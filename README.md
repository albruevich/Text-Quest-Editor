# Text Quest Editor

---

QuestEditor позволяет вам создавать и использовать текстовые квесты, наподобие квестов в игре «Космические рейнджеры».  
QuestEditor представлен в виде готовых сборок для платформ **Windows, macOS, Linux**.

Готовые квесты можно как проигрывать внутри самого QuestEditor для тестирования, так и добавлять их внутрь QuestReader, который является open-source и позволяет видеть и понимать работу квеста, использовать код QuestReader в своих проектах и настраивать UI под себя.

Это два проекта (**QuestEditor + QuestReader**), которые работают в связке.

Квест создаётся и хранится в формате **JSON**, что делает систему универсальной — вы можете использовать свои квесты в любых проектах, даже не обязательно на Unity.

---

## Создание нового квеста

Чтобы создать новый квест:

1. Нажмите на кнопку  
<img src="docs/btn_new.webp" width="46">

2. Введите название квеста → Accept  
3. Название квеста является его **ID** и не изменяется (в отличие от DisplayName)  
4. Название квеста должно совпадать с названием папки квеста (при создании это делается автоматически — главное не менять названия)

## Загрузка квеста

1. Нажмите на кнопку  
<img src="docs/btn_load.webp" width="47">

2. Выберите нужный квест  
3. Нажмите кнопку **Load**

## Сохранение квеста

1. Нажмите на кнопку  
<img src="docs/btn_save.webp" width="44">

2. Нажмите кнопку **Save**  
3. Можно использовать горячие клавиши:  
   - **Ctrl + S** (Windows)  
   - **Cmd + S** (macOS)

## Добавление своих квестов в QuestReader

Чтобы использовать свой квест в QuestReader:

1. Нажмите кнопку сохранения квеста  
<img src="docs/btn_save.webp" width="44">

2. Нажмите **Open folder**, откроется папка `Quests/YourQuest` — скопируйте её  
3. Откройте проект QuestReader через Unity  
4. Вставьте папку вашего квеста по адресу:  
   `Assets/Resources/Quests/`  
5. Добавьте название папки вашего квеста в  
   `Assets/_Settings/Quest Folders List`  
6. Нажмите кнопку **Run** — квест появится в списке

## Структура квеста

Квест представляет собой папку, которая совпадает с названием вашего квеста.

Внутри неё находится:
- основной файл **quest.json**
- папка **Images**
- папка **Sounds**
- папка **Musics**

Все эти папки и файл **quest.json** создаются автоматически при создании квеста.

## Основная мысль / идея

Создание квеста основывается на трёх ключевых понятиях:  
**параметрах, локациях и переходах**.

1. Создаёте параметры  
2. Создаёте локации  
3. Создаёте переходы между локациями  

Дальше всё начинает связываться:

4. Каждая локация и переход может (но не обязательно) влиять на параметры  
5. Показ переходов также может зависеть от параметров  
6. В описаниях и параметрах часто используются формулы, например: `(p1 + 1) / 2`

## Советы

Не беритесь сразу за сложный квест.  
Для начала сделайте небольшой квест на 5–10 локаций.  
Постепенно освоившись, вы сможете добавлять более сложную логику и не запутаетесь в системе.
