# Text Quest Editor

QuestEditor позволяет вам создавать и использовать текстовые квесты, наподобие квестов в игре «Космические рейнджеры».  
QuestEditor представлен в виде готовых сборок для платформ Windows, macOS, Linux.

Готовые квесты можно как проигрывать внутри самого QuestEditor для тестирования, так и добавлять их внутрь QuestReader, который является open-source и позволяет видеть и понимать работу квеста, использовать код QuestReader для своих проектов, настраивать UI под себя.

Это два проекта (QuestEditor + QuestReader), которые работают в связке.

Квест создается и хранится в формате JSON, что делает систему универсальной — можно использовать квесты в любых проектах, даже не обязательно на Unity.

## Основная идея

В основе создания квеста лежат три главные вещи:
параметры, локации и переходы.

Обычно всё строится по такому принципу:

1. Вы создаёте параметры
2. Создаёте локации
3. Соединяете локации переходами

После этого можно постепенно добавлять логику:

4. Локации и переходы могут влиять на параметры
5. От параметров может зависеть, какие переходы будут показаны игроку
6. В описаниях, условиях и значениях параметров часто используются формулы, например: (p1 + 1) / 2

То есть, по сути, весь квест собирается именно из этих трёх элементов.
